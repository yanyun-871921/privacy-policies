PriceScout Privacy Policy

Last updated: 2026-02-02

Summary

PriceScout is a Chrome extension that helps you open search links across multiple marketplaces using a keyword you enter.

Data we collect

  • PriceScout does **not** collect personal data (name, email, phone number).
  • PriceScout does **not** send your browsing data to our servers.

Data stored locally (Chrome sync/storage)

PriceScout may store the following data using Chrome storage to improve user experience:

  • Your last used search keyword
  • Selected marketplaces (checkbox settings)
  • Recent keywords and favorites

This data stays in your browser (and may sync across your Chrome profile if Chrome Sync is enabled).

Permissions

  • activeTab / scripting: used only when you click "페이지에서 찾기" to extract candidate keywords from the current page.
  • storage: used to save your settings and recent keywords.
  • tabs: used to open marketplace search pages in new tabs.
